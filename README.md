# Modern C++ Thread Pool

Linux:

```sh

cd /tmp
git clone https://github.com/johnpatek/thread-pool
mkdir build
cd build
cmake ../thread-pool
make
```

Windows:

```bat

cd %USERPROFILE%\AppData\Local\Temp
git clone https://github.com/johnpatek/thread-pool
mkdir build
cd build
cmake ..\thread-pool
msbuild thread-pool.sln

```
